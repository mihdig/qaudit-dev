## Executive Summary

The digital accessibility market in 2025 is driven by **EAA (European Accessibility Act)** enforcement starting June 28, 2025, with fines up to **€100,000**. QAudit enters a market dominated by enterprise players ($25K-100K/year) with a unique value proposition: **enterprise features at SMB pricing**.

### Key Differentiators

| Feature | QAudit | Competition |
|---------|--------|-------------|
| Tiered Contrast Scanner | Dual screenshot method | Basic computed styles |
| Security + A11Y | Combined scanning | A11Y only |
| Speed | <1 second | 3-5 seconds |
| Price | $49-399/month | $500-100K/year |
| Vision AI | Auto alt-text, fixes | Limited or overlay |

## Unique QAudit Technologies

### 1. Tiered Contrast Scanner

**Problem:** Traditional tools use CSS computed styles, missing:
- Gradient backgrounds
- Image backgrounds
- Semi-transparent overlays
- Text shadows

**QAudit Solution:**
```
TIER 1 (CSS)     → Fast computed style analysis (~2ms)
TIER 2 (Dual SS) → Screenshot with/without text → pixel diff
TIER 3 (Vision)  → AI validation for edge cases
```

**No competitor offers this technology.**

### 2. Security + A11Y Combined

QAudit is the only tool combining:
- 86 WCAG/ACT accessibility rules
- 8 OWASP security checks (CSRF, XSS, mixed content, etc.)

**One scan = complete audit**

### 3. Vision AI Auto-Fixes

- Automatic alt-text generation
- Fix suggestions in natural language
- Code-level remediation (not overlay)
